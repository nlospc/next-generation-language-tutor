# Phase 5: Web UI 壳与内容系统

## 目标

建立本地私有控制台和说明内容入口。

## 交付

- Web server
- Open Design 指导下的轻量 Web UI
- 首页
- 使用说明页面
- 学习方法页面
- 内容从 `content/usage/`、`content/methods/`、`content/agent-playbooks/` 读取
- Markdown/HTML 渲染

## 验证

- 本地访问 `http://127.0.0.1:<port>`。
- 页面稳定、美观、轻量。
- 用户能看到项目说明和学习方法说明。
- agent 能读取同一套内容作为方法论上下文。
