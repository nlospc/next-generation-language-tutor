# Phase 8: Daily Check-in Calendar

## 目标

加入学习打卡反馈机制。

## 交付

- daily status service
- check-in calendar API
- Web UI 月历
- streak 计算
- 今日任务状态：
  - due reviews
  - items added
  - training completed
  - completed/not completed

## 验证

- Web UI 可显示连续学习天数。
- 添加条目或完成复习后更新今日状态。
- MCP `get_daily_status` 能读取同一数据。
